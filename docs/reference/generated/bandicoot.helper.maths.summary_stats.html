summary_stats¶

bandicoot.helper.maths.summary_stats(data)¶

Returns a SummaryStats object containing statistics on the given distribution.

Examples

>>> summary_stats([0, 1])
SummaryStats(mean=0.5, std=0.5, min=0.0, max=1.0, median=0.5, skewness=0.0, kurtosis=1.0, distribution=[0, 1])
